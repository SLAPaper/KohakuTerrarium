# 並排拆分分頁

桌面端儀表板的主區域可以像 VSCode 那樣拆分為多個**分頁群組**。每個群組
都是一條完整的分頁列，擁有各自的作用中分頁，並且可以容納*任意*類型的
分頁——因此你可以讓聊天、程式碼編輯器與儀表板同時並排顯示。

拆分功能**始終可用**——沒有開關或選項。如果你從不拆分，看到的就是一個
分頁群組、一條分頁列（與此功能上線前完全一致），但下述手勢始終生效。

> 拆分適用於桌面/寬螢幕版面。在精簡（行動）外殼中，分頁維持為單一分頁列。

## 拆分分頁群組

- **把分頁拖到任一分頁群組主體的邊緣**——*左 / 右 / 上 / 下* 的 25% 邊緣
  區域——該群組便會沿對應方向拆分，被拖動的分頁落入新建的兄弟群組。拖放
  時會顯示落點高亮。
- **鍵盤：** `Ctrl+\` 將聚焦群組水平拆分（新群組在右側）；`Ctrl+Alt+\`
  垂直拆分（新群組在下方）。

聚焦的群組會有外框；點擊群組內任意位置即可聚焦它。新分頁（`+` 選單，或
開啟一個 creature / 工作階段）會在聚焦群組中開啟。

## 在群組之間移動分頁

- 將分頁拖到**另一個群組的分頁列**，或另一個群組主體的**中心**，即可把
  它移入該群組。
- 在**自身分頁列內**拖動分頁可重新排序。

當某個群組失去最後一個分頁時會自動收合，其兄弟群組擴展填滿空間。

## 調整大小

拖動兩個群組之間的分隔條可改變拆分比例。比例會被限制，任一側都不會低於
10%。

## 關閉

- 用分頁上的 `×`（或滑鼠中鍵）關閉分頁。若因此清空了某個群組，該群組會
  收合。
- `Ctrl+Shift+W` 關閉整個聚焦群組（僅當存在多個群組時——你永遠不會落到
  零個群組）。儀表板永不關閉；若它位於被關閉的群組中，會遷移到存活的
  群組。

## 鍵盤快速鍵

| 按鍵 | 操作 |
| --- | --- |
| `Ctrl+\` | 水平拆分聚焦群組（新群組在右側）。 |
| `Ctrl+Alt+\` | 垂直拆分聚焦群組（新群組在下方）。 |
| `Ctrl+Shift+W` | 關閉聚焦群組（當存在多個群組時）。 |
| `Ctrl+Tab` / `Ctrl+Shift+Tab` | 依樹走訪順序循環聚焦下一個 / 上一個群組。 |
| `Ctrl+1` … `Ctrl+9` | 聚焦第 N 個群組（依樹走訪順序）。 |

當聚焦的是**聊天**分頁時，這些巨觀層級的快速鍵會讓位——聊天分頁以相同
按鍵操作其內部的[多聊天群組](multi-chat.md)。要把聊天分頁拆分到獨立
群組，請把它的分頁拖到邊緣。

## 持久化

你的分頁群組版面會依瀏覽器（localStorage）保存，並在下次造訪時還原——
包含存在哪些群組、它們的拆分比例，以及每個群組中哪個分頁為作用中。版面
刻意不寫入 URL；請將路由加入書籤，而非排列方式。
