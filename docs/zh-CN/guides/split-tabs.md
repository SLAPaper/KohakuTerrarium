# 并排拆分标签页

桌面端仪表盘的主区域可以像 VSCode 那样拆分为多个**标签组**。每个组都是
一个完整的标签栏，拥有各自的活动标签，并且可以容纳*任意*类型的标签——
因此你可以让聊天、代码编辑器和仪表盘同时并排显示。

拆分功能**始终可用**——没有开关或选项。如果你从不拆分，看到的就是一个
标签组、一条标签栏（与该功能上线前完全一致），但下述手势始终生效。

> 拆分适用于桌面/宽屏布局。在紧凑（移动）外壳中，标签保持为单一标签栏。

## 拆分标签组

- **把标签拖到任意标签组主体的边缘**——*左 / 右 / 上 / 下* 的 25% 边缘
  区域——该组就会沿对应方向拆分，被拖动的标签落入新建的兄弟组。拖放时
  会显示落点高亮。
- **键盘：** `Ctrl+\` 将聚焦组水平拆分（新组在右侧）；`Ctrl+Alt+\` 垂直
  拆分（新组在下方）。

聚焦的组会有描边；点击组内任意位置即可聚焦它。新标签（`+` 菜单，或打开
一个 creature / 会话）会在聚焦组中打开。

## 在组之间移动标签

- 将标签拖到**另一个组的标签栏**，或另一个组主体的**中心**，即可把它
  移入该组。
- 在**自身标签栏内**拖动标签可重新排序。

当某个组失去最后一个标签时会自动折叠，其兄弟组扩展填充空间。

## 调整大小

拖动两个组之间的分隔条可改变拆分比例。比例会被限制，任一侧都不会低于
10%。

## 关闭

- 用标签上的 `×`（或鼠标中键）关闭标签。如果由此清空了某个组，该组会
  折叠。
- `Ctrl+Shift+W` 关闭整个聚焦组（仅当存在多个组时——你永远不会落到零个
  组）。仪表盘永不关闭；若它位于被关闭的组中，会迁移到存活的组。

## 键盘快捷键

| 按键 | 操作 |
| --- | --- |
| `Ctrl+\` | 水平拆分聚焦组（新组在右侧）。 |
| `Ctrl+Alt+\` | 垂直拆分聚焦组（新组在下方）。 |
| `Ctrl+Shift+W` | 关闭聚焦组（当存在多个组时）。 |
| `Ctrl+Tab` / `Ctrl+Shift+Tab` | 按树遍历顺序循环聚焦下一个 / 上一个组。 |
| `Ctrl+1` … `Ctrl+9` | 聚焦第 N 个组（按树遍历顺序）。 |

当聚焦的是**聊天**标签时，这些宏观层级的快捷键会让位——聊天标签用相同
按键操作其内部的[多聊天分组](multi-chat.md)。要把聊天标签拆分到独立组，
请把它的标签拖到边缘。

## 持久化

你的标签组布局会按浏览器（localStorage）保存，并在下次访问时恢复——包括
存在哪些组、它们的拆分比例，以及每个组中哪个标签处于活动状态。布局有意
不写入 URL；请收藏路由，而非排布方式。
